PLATAFROMA DE CONTROLDE PEDIDOS, EN PROGRESO

Para mi empresa ,Codibarras & Grupo GAV Ltda , estoy desarrollando una plataforma web que mos permita gestionar los pedidos, analizar los datos con un dashboard y generar reportes. La plataforma permite a los usuarios registrar y gestionar los pedidos de manera eficiente. Se proyecta implemntar Django como backend y VUE JS como frontend. La plataforma también incluye un sistema de autenticación y autorización para garantizar la seguridad de los datos.Y como motor de Base de Datos se usará MariaDB.
